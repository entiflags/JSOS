# jsos
jsos is operating system project written 100% in java script i using node js for building

## how to build this thing
you need a node js just write in a terminal "npm start 'filename'" filename is whatever name you want
be your bootable image
for example: "npm start operatingsystem" and it will create "operatingsystem" image and you just write like
"qemu-system-i386 -hda operatingsystem" and it will run the image in qemu
